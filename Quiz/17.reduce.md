## 17. reduce

### Problem

https://bigfrontend.dev/quiz/reduce

### Problem Description

What does the code snippet below output by `console.log`?

```js
[1, 2, 3].reduce((a, b) => {
  console.log(a, b);
});

[1, 2, 3].reduce((a, b) => {
  console.log(a, b);
}, 0);
```

### Answer

```
1,2
undefined,3
0,1
undefined,2
undefined,3
```
