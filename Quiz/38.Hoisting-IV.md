## 38. Hoisting IV

### Problem

https://bigfrontend.dev/quiz/Hoisting-IV

### Problem Description

What does the code snippet below output by `console.log`?

```js
let foo = 10;
function func1() {
  console.log(foo);
  var foo = 1;
}
func1();

function func2() {
  console.log(foo);
  let foo = 1;
}
func2();
```

### Answer

```
undefined
Error
```
