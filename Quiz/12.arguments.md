## 12. arguments

### Problem

https://bigfrontend.dev/quiz/arguments

### Problem Description

What does the code snippet below output by `console.log`?

```js
function log(a, b, c, d) {
  console.log(a, b, c, d);
  arguments[0] = 'bfe';
  arguments[3] = 'dev';

  console.log(a, b, c, d);
}

log(1, 2, 3);
```

### Solution

```
1,2,3,undefined
"bfe",2,3,undefined
```
