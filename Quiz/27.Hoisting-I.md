# 27. Hoisting I

### Problem

https://bigfrontend.dev/quiz/Hoisting-I

#

### Problem Description

What does the code snippet below output by `console.log`?

```js
const a = 1;
console.log(a);

var b;
console.log(b);
b = 2;

console.log(c);
var c = 3;

console.log(d);
let d = 2;
```

#

### Answer

```js
const a = 1;
console.log(a); // 1

var b;
console.log(b); // undefined
b = 2;

console.log(c); // undefined
var c = 3;

console.log(d); // Error
let d = 2;
```
