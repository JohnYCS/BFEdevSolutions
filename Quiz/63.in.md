## 63. in

### Problem

https://bigfrontend.dev/quiz/in-coercion

### Problem Description

What does the code snippet below output by `console.log`?

```js
const obj = {
  foo: 'bar',
};

console.log('foo' in obj);
console.log(['foo'] in obj);
```

### Solution

```
true
true
```

Non-strings will be coerced to strings.
