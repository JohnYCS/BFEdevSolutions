## 7. Increment Operator

### Problem

https://bigfrontend.dev/quiz/Increment-Operator

### Problem Description

What does the code snippet below output by `console.log`?

```js
let a = 1;
const b = ++a;
const c = a++;
console.log(a);
console.log(b);
console.log(c);
```

### Solution

```
3
2
2
```
